# Browser Wrapper

## Usage

1. Set target browser's intent handler.
2. Set this application to the default browser.

### Incognito Mode Support

#### Chromium

Enable the following flags:

- chrome://flags/#cct-incognito
- chrome://flags/#cct-incognito-available-to-third-party

#### Firefox

Unsupported
